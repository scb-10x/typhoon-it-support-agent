# New Employee IT Onboarding Guide

## Welcome to the Company!

This guide will help you get started with all the IT systems and tools you'll need for your first day and beyond.

## First Day - Morning

### 1. IT Department Check-In

**Location**: IT Department (Floor 3, Room 301)

**What Will Happen:**

1. **Identity Verification**
   - Show your photo ID
   - Sign employment verification form

2. **Receive Your Equipment**
   - Laptop or desktop computer
   - Charger and accessories
   - Mouse and keyboard (if desktop)
   - Headset with microphone
   - Security badge/access card

3. **Sign Asset Agreement**
   - Acknowledge receipt of equipment
   - Understand care responsibilities
   - Note asset tag numbers

### 2. Initial Account Setup

IT will help you with:

**Company Credentials:**
- **Username**: Usually firstname.lastname or employee ID
- **Temporary Password**: You'll need to change this immediately
- **Email**: firstname.lastname@company.com

**Password Requirements:**
- Minimum 12 characters
- Include uppercase, lowercase, numbers, special characters
- Cannot contain your name or "company"
- Must be unique (not used elsewhere)

**Change Your Password:**
1. Press Ctrl + Alt + Delete
2. Select "Change a password"
3. Enter temporary password
4. Enter new password twice
5. Remember this password - you'll need it frequently!

### 3. Multi-Factor Authentication (MFA) Setup

MFA adds security by requiring your phone in addition to password.

**Setup Steps:**

1. **Install Authenticator App**
   - Download Microsoft Authenticator (recommended) or Google Authenticator
   - Available on iOS App Store or Google Play

2. **Configure MFA**
   - IT will take you to: https://aka.ms/mfasetup
   - Click "Add method" > "Authenticator app"
   - Scan QR code with your phone
   - Enter 6-digit code shown in app

3. **Add Backup Methods**
   - Add mobile phone number (for SMS backup)
   - Add alternate phone (optional)

4. **Save Backup Codes**
   - You'll receive 10 backup codes
   - Print or write these down
   - Store in a safe place
   - Use if you lose your phone

**Important:** Don't leave IT without successfully setting up MFA and testing login!

### 4. Computer Setup

IT will configure your computer with:

**Operating System:**
- Windows 10/11 Professional
- Or macOS (if approved for your role)
- Fully updated with latest patches

**Pre-installed Software:**
- Microsoft Office 365 (Word, Excel, PowerPoint, Outlook)
- Microsoft Teams
- OneDrive for Business
- Web browsers (Edge, Chrome)
- Adobe Acrobat Reader
- Zoom or video conferencing tools
- Antivirus software
- VPN client (Cisco AnyConnect)
- Company-specific applications

**Settings Configured:**
- Connected to company domain
- Email profile set up
- Network drives mapped
- Printer access configured
- Security policies applied
- Encryption enabled

### 5. Test Your Access

Before leaving IT, test:

- [ ] Login to computer successfully
- [ ] Open Outlook and see email
- [ ] Access company intranet
- [ ] Connect to WiFi: "CompanySecure"
- [ ] Print a test page
- [ ] Access network drives

## First Day - Afternoon

### 6. Set Up Your Workspace

**Physical Setup:**
1. Connect all cables (power, monitor, network)
2. Arrange monitor at eye level
3. Position keyboard and mouse comfortably
4. Connect headset
5. Organize desk area

**Computer Personalization:**
- Set comfortable screen brightness
- Adjust volume
- Customize desktop background (keep professional)
- Configure taskbar shortcuts
- Set up any accessibility needs

### 7. Configure Microsoft Outlook

**First Launch:**
1. Open Outlook
2. It should auto-configure with your email
3. Wait for initial email download
4. Review Focused Inbox vs Other tabs

**Set Up Your Signature:**
1. File > Options > Mail > Signatures
2. Click "New"
3. Create signature with:
   ```
   [Your Name]
   [Your Job Title]
   [Department]
   [Phone Extension]
   yourname@company.com
   ```
4. Set as default for new messages and replies

**Configure Out of Office (for future):**
- Remember: File > Automatic Replies
- You'll use this for vacation and sick days

**Add Your Manager's Calendar:**
1. Calendar view > Add Calendar
2. From Address Book
3. Select your manager
4. Helps schedule meetings

### 8. Set Up Microsoft Teams

**First Launch:**
1. Sign in with company credentials
2. Complete quick tutorial if offered
3. Upload professional profile photo

**Join Your Team:**
- Your manager should have added you to relevant Teams
- Check Teams list on left sidebar
- Join "Company All Hands" or similar company-wide teams

**Set Your Status:**
- Click your profile picture
- Set status to "Available"
- Add status message if desired

**Test Audio/Video:**
1. Settings (gear icon) > Devices
2. Test microphone
3. Test speakers
4. Test camera
5. Make a test call to Audio Test Service

### 9. OneDrive Setup

**Initial Sync:**
1. OneDrive should be running (cloud icon in taskbar)
2. If not: Start > OneDrive
3. Sign in with company credentials
4. Choose sync location (default is fine)
5. Select "Sync all files and folders" or choose specific folders

**Understand the Folder:**
- Your OneDrive folder: C:\Users\[YourName]\OneDrive - Company
- Save work files here, NOT in Documents
- Files sync automatically to cloud
- Access from any device

**Files On-Demand:**
- Already enabled by default
- Files don't take up space until you open them
- Cloud icon means stored online only
- Green check means downloaded locally

### 10. Learn the Network Drives

Your computer should have these mapped:

**Common Drives:**
- **H: Drive** - Your personal home drive (just for you)
- **S: Drive** - Shared departmental drive
- **T: Drive** - Company-wide shared drive

**Where to Save What:**
- **OneDrive**: Your work documents, presentations, spreadsheets
- **SharePoint/Teams**: Team collaboration files
- **H: Drive**: Personal work files (if needed)
- **S: Drive**: Department shared files
- **Never save to**: Desktop or local C: drive (not backed up!)

**IT Portal** (self-service)
- **URL**: https://itportal.company.com

### How to Submit a Ticket

**Via Portal:**
1. Go to IT Portal
2. Click "Submit a Ticket"
3. Choose category (Hardware, Software, Access, etc.)
4. Describe issue clearly
5. Add screenshots if helpful
6. Select priority
7. Submit

**Via Email:**
1. Email helpdesk@company.com
2. Clear subject line
3. Detailed description
4. What you tried already
5. Your phone number

**Via Phone:**
- Call ext. 1234
- Explain issue
- Ticket created automatically
- Get ticket number for tracking

### What Information to Provide

When requesting help:
- Your name and employee ID
- Computer name or asset tag
- Detailed description of problem
- Error messages (exact text or screenshot)
- When did it start?
- What were you doing?
- What have you tried?
- How urgent is it?

## Common First Week Questions

### Q: Can I install software myself?
**A:** From Software Center, yes! Other software needs IT approval for security reasons.

### Q: Can I use personal email on my work computer?
**A:** Via web browser is okay. Don't configure personal email in Outlook.

### Q: What if I forget my password?
**A:** Go to https://itportal.company.com > "Reset Password" or call IT helpdesk.

### Q: Can I take my laptop home?
**A:** Yes! Laptops are for your use. See Remote Work Guide for VPN setup.

### Q: How do I connect to WiFi?
**A:** 
- Network: CompanySecure
- Credentials: Your company username and password
- Should connect automatically after first time

### Q: What if I need a second monitor?
**A:** Submit request through IT Portal > Hardware Request. Most roles eligible.

### Q: How do I access company resources from home?
**A:** Connect via VPN first. See Remote Work Guide.

### Q: My email quota is filling up. What do I do?
**A:** Archive old emails or delete unnecessary ones. IT can help enable archive mailbox.

### Q: How do I print?
**A:** Printers should be pre-configured. Find yours at http://printserver.company.com

### Q: What if equipment is damaged or broken?
**A:** Report immediately to IT helpdesk. Don't try to fix it yourself.

## Security Reminders

### Do's

- ✅ Lock your computer when leaving desk (Windows + L)
- ✅ Use strong, unique passwords
- ✅ Enable MFA on all accounts
- ✅ Save files to OneDrive/network drives
- ✅ Report suspicious emails to security@company.com
- ✅ Keep software updated
- ✅ Use VPN when working remotely
- ✅ Encrypt sensitive data before sharing
- ✅ Report lost/stolen equipment immediately
- ✅ Complete security training on time

### Don'ts

- ❌ Share your password with anyone (not even IT!)
- ❌ Leave computer unlocked
- ❌ Click links in suspicious emails
- ❌ Install unauthorized software
- ❌ Save work files to personal cloud storage
- ❌ Use simple or common passwords
- ❌ Connect to public WiFi without VPN
- ❌ Share sensitive data via unsecured methods
- ❌ Let others use your computer
- ❌ Disable antivirus or security software

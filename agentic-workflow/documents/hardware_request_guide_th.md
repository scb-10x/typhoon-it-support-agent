# คู่มือการขออุปกรณ์คอมพิวเตอร์และฮาร์ดแวร์

## ภาพรวม

เอกสารนี้อธิบายขั้นตอนการขออุปกรณ์คอมพิวเตอร์ อุปกรณ์เสริม และการอัพเกรดฮาร์ดแวร์

---

## 1. อุปกรณ์มาตรฐานสำหรับพนักงาน

### พนักงานทั่วไป
- **Laptop/Desktop**: Dell Latitude หรือ HP ProBook
- **Monitor**: 21-24 นิ้ว (ถ้าใช้ Desktop)
- **คีย์บอร์ดและเมาส์**: USB มาตรฐาน
- **Headset**: USB Headset พื้นฐาน
- **USB Hub**: 4 พอร์ต (ถ้าจำเป็น)

### พนักงาน Development/Design
- **Laptop**: Dell Precision หรือ MacBook Pro
- **RAM**: ขั้นต่ำ 16GB
- **Monitor**: 27 นิ้ว หรือ Dual Monitors
- **อุปกรณ์เสริม**: Mechanical Keyboard, Ergonomic Mouse
- **Graphics Tablet**: สำหรับ Designer

### Management Level
- **Laptop**: Business Class (Dell XPS, ThinkPad X1, MacBook Pro)
- **Docking Station**: Universal Dock
- **Monitor**: 27 นิ้ว 4K
- **อุปกรณ์ประชุม**: Webcam HD, Headset พรีเมี่ยม

## 2. วิธีการขออุปกรณ์

### ขั้นตอนการขอ

#### Step 1: เข้าสู่ IT Portal
1. เปิดเบราว์เซอร์ไปที่ https://itportal.company.com
2. Login ด้วย Username และ Password ของบริษัท
3. เลือก "Hardware Request" จากเมนู

#### Step 2: กรอกฟอร์มคำขอ

**ข้อมูลที่ต้องกรอก:**

1. **ประเภทคำขอ**:
   - อุปกรณ์ใหม่สำหรับพนักงานใหม่
   - เปลี่ยนอุปกรณ์เดิมที่เสีย
   - อัพเกรดอุปกรณ์เดิม
   - อุปกรณ์เสริม

2. **รายละเอียดอุปกรณ์**:
   - ประเภทอุปกรณ์ (Laptop, Monitor, etc.)
   - Specification ที่ต้องการ
   - ยี่ห้อ/รุ่น (ถ้าระบุได้)
   - จำนวน

3. **เหตุผล**:
   - อธิบายว่าทำไมต้องการอุปกรณ์นี้
   - อธิบายการใช้งาน
   - ถ้าเป็นการอัพเกรด อธิบายว่าอุปกรณ์เดิมไม่เพียงพอยังไง

4. **รายละเอียดการเงิน**:
   - Cost Center
   - Budget Code
   - ผู้อนุมัติ

5. **ความเร่งด่วน**:
   - Normal (2-4 สัปดาห์)
   - Urgent (1 สัปดาห์) - ต้องมีเหตุผล
   - Emergency (24-48 ชั่วโมง) - ต้องได้รับอนุมัติพิเศษ

#### Step 3: แนบเอกสาร (ถ้ามี)
- ใบอนุมัติจากหัวหน้างาน
- เอกสารประกอบการขออัพเกรด
- Screenshot ปัญหาของอุปกรณ์เดิม
- ใบเสนอราคา (ถ้าเป็นอุปกรณ์พิเศษ)

#### Step 4: Submit
1. ตรวจสอบข้อมูลทั้งหมด
2. กดปุ่ม "Submit Request"
3. จดเลข Ticket Number สำหรับติดตาม
4. ระบบจะส่งอีเมลยืนยันมาให้

## 3. Process การอนุมัติ

### ขั้นตอนการอนุมัติ

```
[คำขอ] → [IT Review] → [Manager Approval] → [Budget Approval] → [Procurement] → [Delivery]
```

**1. IT Review (1-2 วัน)**
- IT ตรวจสอบความเหมาะสม
- เช็ค Specification
- แนะนำทางเลือก (ถ้ามี)

**2. Manager Approval (2-3 วัน)**
- หัวหน้างานอนุมัติ
- ตรวจสอบความจำเป็น
- อนุมัติงบประมาณ

**3. Budget Approval (1-2 วัน)**
- Finance Department ตรวจสอบงบประมาณ
- ยืนยัน Cost Center
- อนุมัติการจัดซื้อ

**4. Procurement (1-2 สัปดาห์)**
- จัดซื้อจากผู้ขาย
- ตรวจสอบสินค้า
- เตรียมอุปกรณ์

**5. Delivery & Setup (1-2 วัน)**
- แจ้งพนักงานมารับอุปกรณ์
- IT ติดตั้งและ Configure
- ทดสอบการทำงาน
- มอบอุปกรณ์พร้อมเอกสาร

### เวลาโดยประมาณ

| ประเภท | เวลาดำเนินการ |
|--------|---------------|
| อุปกรณ์มาตรฐาน | 2-3 สัปดาห์ |
| อุปกรณ์พิเศษ | 3-6 สัปดาห์ |
| อุปกรณ์เสริม | 1-2 สัปดาห์ |
| อุปกรณ์ในคลัง | 2-3 วัน |
| Emergency | 24-48 ชั่วโมง |

## 5. การขออัพเกรดอุปกรณ์เดิม

### กรณีที่อนุญาตให้อัพเกรด

**1. อายุการใช้งาน:**
- Desktop/Laptop: มากกว่า 3 ปี
- Monitor: มากกว่า 5 ปี
- Accessories: เมื่อเสีย/ไม่เพียงพอ

**2. ประสิทธิภาพ:**
- ระบบช้ามาก แม้ได้ทำ Maintenance แล้ว
- ไม่รองรับซอฟต์แวร์ที่จำเป็น
- Hardware ล้าสมัยเกินไป

**3. ความจำเป็นจากงาน:**
- เปลี่ยนตำแหน่งงาน
- งานใหม่ต้องใช้ Spec สูงขึ้น
- รับผิดชอบงานพิเศษ

### Component ที่อัพเกรดได้

**Laptop/Desktop:**
- ✅ RAM (เพิ่มความจำ)
- ✅ Storage (เพิ่มฮาร์ดดิสก์/SSD)
- ❌ CPU (ต้องเปลี่ยนเครื่องใหม่)
- ❌ Motherboard (ต้องเปลี่ยนเครื่องใหม่)

**เพิ่มอุปกรณ์เสริม:**
- ✅ Monitor เพิ่ม (สำหรับ Developer/Designer)
- ✅ Docking Station
- ✅ External Storage
- ✅ Webcam/Headset คุณภาพดีขึ้น

### วิธีการขออัพเกรด

1. **เตรียมข้อมูล:**
   - ตรวจสอบ Specification ปัจจุบัน
   - Screenshot ปัญหาประสิทธิภาพ (Task Manager, etc.)
   - อธิบายว่าทำงานอะไรที่ต้องใช้ Spec สูง

2. **สร้างคำขอ:**
   - ไปที่ IT Portal > Hardware Request
   - เลือก "Upgrade Existing Equipment"
   - กรอกรายละเอียดปัจจุบันและที่ต้องการ

3. **Justification:**
   - อธิบายชัดเจนว่าทำไมต้องอัพเกรด
   - แนบหลักฐาน (screenshot, work requirements)
   - ประมาณการประโยชน์ที่ได้รับ

4. **รอการพิจารณา:**
   - IT จะประเมินว่าควรอัพเกรดหรือเปลี่ยนเครื่องใหม่
   - อาจมีการนัดคุยเพิ่มเติม
   - ใช้เวลา 1-2 สัปดาห์

## 6. อุปกรณ์พิเศษ

### อุปกรณ์ที่ต้องการอนุมัติพิเศษ

**High-End Equipment:**
- MacBook Pro (ถ้าไม่ใช่ตำแหน่งที่กำหนดไว้)
- Gaming/High-Performance Laptop
- Workstation Grade Desktop
- Monitor 4K ขนาดใหญ่ (32 นิ้วขึ้นไป)
- Professional Camera/Microphone

**Specialized Equipment:**
- Graphics Tablet (Wacom Cintiq)
- 3D Mouse
- Professional Audio Interface
- Drawing Tablet
- Ergonomic Chair (ต้องมีใบรับรองแพทย์)

### ขั้นตอนการขออุปกรณ์พิเศษ

1. **ปรึกษา IT ก่อน:**
   - อธิบายความต้องการ
   - IT จะแนะนำทางเลือก
   - ประเมินราคาคร่าวๆ

2. **เตรียมเอกสารประกอบ:**
   - เหตุผลการใช้งานอย่างละเอียด
   - ทำไมอุปกรณ์มาตรฐานไม่เพียงพอ
   - ROI (Return on Investment) ถ้าเป็นอุปกรณ์แพง
   - ใบอนุมัติจากหัวหน้าแผนก

3. **Submit Request:**
   - ผ่าน IT Portal แบบปกติ
   - แนบเอกสารประกอบครบถ้วน
   - ระบุ "Special Equipment" ในหัวข้อ

4. **รอการพิจารณา:**
   - IT + Manager + Finance จะประชุมพิจารณา
   - อาจต้องนำเสนอเพิ่มเติม
   - ใช้เวลา 2-4 สัปดาห์

## 7. การรับและตรวจสอบอุปกรณ์

### เมื่อได้รับแจ้งให้มารับอุปกรณ์

**เตรียมสิ่งเหล่านี้:**
- บัตรพนักงาน
- Ticket Number
- อุปกรณ์เดิม (ถ้าเป็นการเปลี่ยน)
- เอกสารที่เกี่ยวข้อง

### ขั้นตอนการรับอุปกรณ์

**1. ยืนยันตัวตน**
- แสดงบัตรพนักงาน
- ยืนยัน Ticket Number

**2. ตรวจสอบอุปกรณ์**
- ✅ ตรงตาม Spec ที่ขอหรือไม่
- ✅ ไม่มีรอยเสียหาย
- ✅ อุปกรณ์เสริมครบถ้วน (Charger, Cable, etc.)
- ✅ มี Warranty Card และ Manual

**3. IT Setup และ Configure**
- IT จะติดตั้งระบบปฏิบัติการ
- ติดตั้งซอฟต์แวร์มาตรฐาน
- Configure ให้เชื่อมต่อ Domain
- ติดตั้ง Antivirus และ Security Software
- Login ด้วย Username ของคุณเพื่อทดสอบ

**4. Testing**
- ทดสอบการ Login
- ทดสอบการเชื่อมต่อเครือข่าย
- ทดสอบการเข้าถึง Email และ Shared Drive
- ทดสอบอุปกรณ์เสริม (Monitor, Keyboard, Mouse)

**5. ลงนามรับอุปกรณ์**
- เซ็นต์ใน IT Asset Management Form
- รับเอกสาร:
  - Warranty Card
  - User Manual
  - IT Support Contact Card
  - Asset Tag Number

**6. ส่งคืนอุปกรณ์เดิม**
- ถ้าเป็นการเปลี่ยน ส่งอุปกรณ์เดิมให้ IT
- ถอน Asset Tag ออก
- เซ็นต์ส่งมอบในแบบฟอร์ม

## 8. การดูแลรักษาอุปกรณ์

### ความรับผิดชอบของผู้ใช้งาน

**DO's:**
- ✅ ใช้งานตามวัตถุประสงค์
- ✅ ทำความสะอาดเป็นประจำ
- ✅ เก็บในที่ปลอดภัย
- ✅ Restart เครื่องอย่างน้อยสัปดาห์ละครั้ง
- ✅ อัพเดท Windows และซอฟต์แวร์
- ✅ Backup ข้อมูลสำคัญ
- ✅ รายงานปัญหาทันทีที่พบ

**DON'Ts:**
- ❌ ติดตั้งซอฟต์แวร์ที่ไม่ได้รับอนุญาต
- ❌ ถอด/เปลี่ยน Component เอง
- ❌ เปิดฝาเครื่อง
- ❌ นำไปซ่อมที่อื่นโดยไม่ได้รับอนุญาต
- ❌ ปล่อยให้คนอื่นใช้งาน
- ❌ วางของหนักทับ Laptop
- ❌ กินอาหาร/ดื่มน้ำข้างๆ อุปกรณ์

### การรายงานปัญหา

**เมื่ออุปกรณ์เสีย/มีปัญหา:**
1. อย่าพยายามซ่อมเอง
2. สร้าง Ticket ที่ IT Portal
3. อธิบายปัญหาอย่างละเอียด
4. แนบรูปถ้ายืนเหตุการณ์ (ถ้ามี)
5. ระบุความเร่งด่วน
6. รอคำตอบจาก IT

**สำหรับปัญหาฉุกเฉิน:**
- โทรหา Helpdesk: ext. 1234
- อธิบายปัญหา
- IT จะจัดลำดับความสำคัญ

## 9. การส่งคืนอุปกรณ์

### กรณีที่ต้องส่งคืน

- ลาออกจากบริษัท
- เปลี่ยนตำแหน่งที่ไม่ต้องใช้อุปกรณ์นี้
- ได้รับอุปกรณ์ใหม่มาแทน
- อุปกรณ์ชำรุด/เสียเกินซ่อม

### ขั้นตอนการส่งคืน

**1. แจ้งความประสงค์**
- สร้าง Ticket "Return Equipment"
- ระบุเหตุผล
- นัดวันเวลา

**2. เตรียมอุปกรณ์**
- Backup ข้อมูลส่วนตัว
- ลบข้อมูลส่วนตัว (รูปภาพ, เอกสาร)
- ทำความสะอาด
- รวบรวมอุปกรณ์เสริม (Charger, Cable, Bag)
- หา Asset Tag Number

**3. ส่งมอบที่ IT Department**
- นำอุปกรณ์ทั้งหมดมา
- นำเอกสาร Warranty และ Manual มาด้วย
- IT จะตรวจสอบอุปกรณ์
- ลงนามส่งมอบ

**4. รับใบรับรอง**
- รับเอกสารยืนยันการส่งคืน
- เก็บไว้สำหรับบันทึก
- ถ้าเป็นการลาออก แนบกับเอกสาร Exit Interview

### การจัดการข้อมูล

**ก่อนส่งคืน:**
- ✅ Backup ข้อมูลไปที่ OneDrive/Network Drive
- ✅ Export Outlook Data (PST file) ถ้าจำเป็น
- ✅ Export Bookmarks จากเบราว์เซอร์
- ✅ ถ่ายรูปการตั้งค่าสำคัญ

**IT จะทำ:**
- Secure Wipe ข้อมูลทั้งหมด
- ตรวจสอบสภาพเครื่อง
- จัดเก็บหรือ Refurbish เพื่อใช้ต่อ

## 10. FAQ

### Q1: ใช้เวลานานแค่ไหนถึงจะได้รับอุปกรณ์?
**A:** ขึ้นอยู่กับประเภท:
- อุปกรณ์ในคลัง: 2-3 วัน
- อุปกรณ์มาตรฐาน: 2-3 สัปดาห์
- อุปกรณ์พิเศษ: 3-6 สัปดาห์

### Q2: สามารถเลือกยี่ห้อและรุ่นเองได้ไหม?
**A:** ได้บางกรณี แต่ต้องเป็นอุปกรณ์ที่บริษัทอนุมัติและอยู่ในงบประมาณ

### Q3: ถ้าอุปกรณ์เสียภายใน Warranty จะทำอย่างไร?
**A:** 
1. สร้าง Ticket แจ้ง IT
2. IT จะจัดการเคลมกับผู้ขาย
3. คุณอาจได้อุปกรณ์ชั่วคราวระหว่างซ่อม

### Q4: สามารถนำอุปกรณ์กลับบ้านได้ไหม?
**A:** 
- Laptop: ได้ (เพื่อ Remote Work)
- Desktop: ไม่ได้ (ยกเว้นได้รับอนุญาต)
- อุปกรณ์เสริม: พิจารณาเป็นรายกรณี

### Q5: ถ้าอุปกรณ์สูญหายหรือเสียเพราะความประมาท?
**A:** 
- ต้องรายงานทันที
- อาจต้องชดใช้ค่าเสียหาย (ขึ้นอยู่กับ Policy)
- ต้องทำรายงานเหตุการณ์

### Q6: สามารถใช้อุปกรณ์ส่วนตัวทำงานได้ไหม?
**A:** 
- BYOD (Bring Your Own Device) อนุญาตบางกรณี
- ต้องติดตั้ง MDM (Mobile Device Management)
- ต้องปฏิบัติตาม Security Policy
- ติดต่อ IT เพื่อลงทะเบียน

### Q7: สามารถขอ Multiple Monitor ได้ไหม?
**A:** 
- Developer/Designer: ได้ (ตามมาตรฐาน)
- ตำแหน่งอื่นๆ: ต้องขออนุมัติพร้อมเหตุผล
- บางแผนกอนุญาต 2 monitors
